API Documentation

RESTful API for programmatic access to NHANES analyses

Base URL: http://localhost:8501 (local) or https://api.nhanes-lancet.com (production)
POST/api/projects Create a new analysis project
Parameters (multipart/form-data)
NameTypeRequiredDescription
titlestringYesProject title
descriptionstringNoProject description
topicstringNoResearch topic for variable mapping
exposurestringNoExposure variable (e.g., BMXBMI)
outcomestringNoOutcome variable (e.g., BPQ020)
cyclesstringNoComma-separated cycles (default: 2017-2018)
analysis_typestringNocross_sectional | cohort | case_control
proposal_filefileNoWord document research proposal
Example
curl -X POST http://localhost:8501/api/projects \
  -F "title=BMI and Hypertension" \
  -F "exposure=BMXBMI" \
  -F "outcome=BPQ020" \
  -F "cycles=2017-2018" \
  -F "analysis_type=cross_sectional"
Response
{
  "project_id": "proj_20260427_203000",
  "status": "created"
}
POST/api/projects/{project_id}/run

Start the analysis pipeline for a created project.

curl -X POST http://localhost:8501/api/projects/proj_20260427_203000/run
{
  "project_id": "proj_20260427_203000",
  "status": "running"
}
GET/api/projects/{project_id}/status
{
  "project_id": "proj_20260427_203000",
  "status": "completed",
  "progress": 100
}
GET/api/variables/search?q={query}&limit={n}

Search NHANES variables by keyword.

curl "http://localhost:8501/api/variables/search?q=blood+pressure&limit=5"
[
  {
    "code": "BPXSY1",
    "description": "Systolic BP - 1st reading",
    "category": "Blood Pressure",
    "type": "continuous",
    "score": 2.0
  },
  ...
]
GET/api/variables/phenotype/{phenotype}

Get recommended variables for a research phenotype. Available: obesity, diabetes, hypertension, dyslipidemia, cardiovascular_disease, smoking, depression, chronic_kidney_disease, diet_quality, sleep

curl http://localhost:8501/api/variables/phenotype/obesity
