Supported Analyses
  • Survey-weighted descriptive statistics
  • Logistic regression (OR, 95% CI)
  • Linear regression (Beta, SE)
  • Kaplan-Meier survival analysis
  • Cox proportional hazards (HR)
  • Fine-Gray competing risk model
  • Subgroup and interaction analyses
  • Sensitivity analyses (multiple imputation)
Lancet-Standard Output
  • Structured abstract (Background/Methods/Findings/Interpretation)
  • Table 1: Baseline characteristics
  • Table 2: Multivariable regression results
  • Forest plots with OR/HR and 95% CI
  • Kaplan-Meier survival curves
  • STROBE checklist compliance
  • Vancouver-format references
  • 300 DPI publication-ready figures
